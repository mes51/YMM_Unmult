# Unmult

## is何

* Knoll UnMultと同様な動きをするエフェクトです
* 黒背景、または白背景に重ねられた画像からα値を復元します

## ダウンロード

* Releasesからymmeファイルをダウンロードしてください

## インストール

* 拡張子の関連付けを行っている場合は、ymmeファイルをダブルクリックします
* 関連付けを行っていない場合はYukkuriMovieMaker.exeにドラッグ&ドロップします
    * もしくは拡張子をzipに変更、解凍したディレクトリをpluginディレクトリそのままコピーします

### 旧版(v1.0.0)からバージョンアップする場合

* YMM_Unmult.dllがpluginディレクトリに残っている場合、先に削除してください

## アンインストール

* pluginディレクトリからYMM_Unmultディレクトリを削除します

## 使用方法

* 映像エフェクトの加工からUnmultを適用します

### パラメータ

* 背景色
    * 何色の背景に重ねられたものとして処理するかを指定します

### 注意点等(使用できる素材等)

* https://www.nicovideo.jp/watch/sm39935396
    * AVIUtl版のものですが動作は全く一緒です
